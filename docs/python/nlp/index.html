    3. nltk: Natural Language Toolkit Docs - good for tokenization, stemming, tagging, parsing, corpora, etc.
    4. spacy: Efficient and Backed by ANNs NLP Toolkit Docs - good for parsing, tagging, entity recognition, text categorization, phrase matching, etc.
